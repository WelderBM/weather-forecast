<h2 id="como-rodar">How to run the project</h2>

``` bash
    # Clone this repository
    $ git clone linkrepo

    # Access the project folder in your terminal
    $ cd weather-forecast

    # Install the dependencies
    $ npm install
    # or
    $ yarn

    # Run the application
    $ npm run dev
    # or
    $ yarn dev
```
